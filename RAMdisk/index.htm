RAMdisk

there and back again

Antec was set up with Ultra RAMDisk
,
  to reduce SSD thrashing and speed Covid calculations.

Switched to ImDisk when Ultra RAMDisk was monetized,
  but Windows 11 changes earlier this year
  disabled launching binaries from ImDisk, then Arduino 2 refused to run.

SCSI miniport

ImDisk uses Direct-IO, which can be faster than SCSI miniport RAMdisks,
  but Arduino 2 will not launch.
Free SCSI miniport RAMdisks:  Arsenal, OFSMount, Ultra RAMDisk, StarWind.
Arsenal is well regarded, but by itself supports neither
  custom drive letter nor Temp folder creation when Windows boots.

tmcdos/ramdisk offers ImDisk functionality for Arsenal.

Back to Ultra RAMDisk Lite

ImDisk lacks uninstaller

  • unmount
  • unclick all options
  • set size to 0
  • set drive to Z
  • not there after reboot...

Ultra RAMDisk Lite install

  • first verify as 100MB S drive
  • remove TEMP folder;  add Temp folder
    (match Chrome browser DATA_CACHE)
  • 2000MB R drive
  • Arduino 2 now launches
  • Decent speed (should have measured ImDisk before removing)
